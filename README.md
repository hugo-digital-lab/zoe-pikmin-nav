# 🛰️ 座標定位分析 (Zoe Pikmin Navigator)

這是一款專為《Pikmin Bloom》與地理定位遊戲打造的輕量級輔助儀表板。由 **Hugo** 構思策劃，並透過專屬 AI **Zoe** 協作開發，旨在解決跨時區座標分析的繁瑣流程，提升資源獲取效率。

## ✨ 核心功能 (Features)

- **自動淨化輸入 (Smart Extraction)**：
  支援直接貼上包含雜訊的群組對話 (如：「大火菇 47.6938, 11.2197 剩2位」)，系統會透過正則表達式自動萃取經緯度，省去手動刪改的麻煩。
- **精準時區對照 (Timezone & Reset Tracking)**：
  自動推算目標座標的當地時間與換日狀態。精準標示「✨ 剛換日 (黃金時段)」，幫助玩家最大化利用時差紅利獲取每日配額。
- **冷卻安全警報 (Cooldown Advisor)**：
  內建距離防呆機制，跨國座標自動亮起紅燈，提醒執行 2 小時的標準冷卻時間，保護帳號安全。
- **一鍵雷達勘景 (Google Maps Integration)**：
  一鍵開啟 Google 衛星地圖，進場前預先判定周邊飾品資源 (如水邊、公園、電影院)。
